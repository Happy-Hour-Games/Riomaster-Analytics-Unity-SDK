# Changelog

## [1.0.0] - 2026-02-18

### Added
- Core analytics tracking (Track, TrackValue)
- Session management (auto session ID, SetPlayerId)
- Progression tracking (LevelStart, LevelComplete, LevelFail)
- Economy tracking (CurrencyEarned, CurrencySpent, ItemAcquired)
- Error tracking (TrackError, TrackException)
- UI interaction tracking
- Tutorial step tracking
- Automatic event batching with configurable interval
- Queue overflow protection
- Automatic flush on app pause/quit
- ScriptableObject configuration support
- Editor window with live stats and quick setup
- UPM (Package Manager) support via Git URL
